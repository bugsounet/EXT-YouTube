# Installing:

```sh
cd ~/MagicMirror/modules/
git clone -b dev https://github.com/bugsounet/MMM-YouTube
```
(npm install is not needed at this day !)


# Configuration:
```js
{
  module: "MMM-YouTube",
  config: {
    fullscreen: false,
    videoID: "RBSZH42a-Kw", // Big Wild - Joypunks
    // "sOnqjkJTMaA" thriller
  }
},
```
